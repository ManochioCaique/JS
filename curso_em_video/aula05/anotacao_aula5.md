# Anotações aula 5

Como fazer comentários:

**//**    - Usados para fazer comentários de um linha

**/*  */** - Usado para fazer comentários com mais de uma linha

- Só funciona quando usamos dentro da parte de JS

# Variáveis

- Variáveis servem para guardar dados

- Nomear variáveis:

      - não podem começar com números
      - é possil ter letras e números
      - è possível usar acentos e simbolos
      - não podem conter espaços
      - não podem ser palavras reservadas

- Existe três tipos de aspas para escrever string:

               " "
               ' '
               `  `
